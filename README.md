# Word Counter & Text Analyzer

## 📌 Overview

This is a Python-based text analysis tool that processes user input and provides detailed statistics about the text including word count, sentence count, character count, and word frequency analysis.

This project demonstrates fundamental Python concepts including:

- String manipulation
- Dictionaries
- Loops
- Functions
- Data analysis logic

---

## 🚀 Features

- Count total words
- Count total characters
- Count total sentences
- Count total paragraphs
- Find most repeated word
- Find longest word
- Calculate average word length

---

## 🛠️ Technologies Used

- Python 3

## ▶️ How to Run

1. Clone repository

```bash
git clone https://github.com/IshaqRiaz/word-counter-text-analyzer.git
```

2. Open project folder

```bash
cd word-counter-text-analyzer
```

3. Run program

```bash
python src/main.py
```
