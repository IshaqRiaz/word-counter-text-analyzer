# Word Counter & Text Analyzer

## 📌 Overview
This project is a Python-based text analysis tool that takes user input and provides detailed insights such as word count, sentence count, character count, paragraph count, and word frequency analysis.

It demonstrates fundamental Python programming concepts including string manipulation, loops, and dictionary usage.

---

## 🚀 Features
- Count total words
- Count total characters
- Count sentences
- Count paragraphs
- Find most common word
- Identify longest word
- Calculate average word length

---

## 🛠️ Technologies Used
- Python 3
